# Kairos

Kairos — интерактивный лендинг с видео-фоном, модальными окнами, формой входа и виджетом криптовалютного рынка. Проект собран на Vite и использует чистый JavaScript, HTML и CSS.

## Возможности

- Адаптивная главная секция с видео-фоном.
- Мобильное бургер-меню.
- Модальные окна для блока `Learn More` и видеоплеера.
- Переключение вкладок формы входа.
- Авторизация через Google OAuth2 с получением профиля пользователя.
- Получение актуальных цен криптовалют через Binance API.
- Сборка статического сайта в `dist`.

## Стек

- Vite
- Vanilla JavaScript
- HTML5
- CSS3

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

После запуска откройте ссылку, которую покажет Vite, обычно:

```text
http://localhost:5173
```

## Сборка

```bash
npm run build
```

Собранный проект появится в папке `dist`.

## Предпросмотр production-сборки

```bash
npm run preview
```

## Структура проекта

```text
kairos/
├─ index.html
├─ package.json
├─ public/
├─ src/
│  ├─ assets/
│  │  ├─ css/
│  │  ├─ fonts/
│  │  └─ images/
│  └─ main.js
└─ dist/
```

## Настройка Google OAuth2

Для работы кнопки Google авторизации замените `client_id` в `src/main.js` на идентификатор вашего OAuth-приложения из Google Cloud Console.

## Примечания

- `node_modules` и `dist` игнорируются Git через `.gitignore`.
- Данные криптовалют загружаются с публичного API Binance и обновляются каждые 5 секунд.
- Для production-деплоя достаточно загрузить содержимое папки `dist` на статический хостинг.
